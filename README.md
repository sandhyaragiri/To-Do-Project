# To-Do-Project

A simple and interactive To-Do App built with HTML, CSS, and JavaScript to practice DOM manipulation and event handling.

## 🚀 Features

* Add new tasks
* Add tasks using the Enter key
* Mark tasks as completed
* Strike-through completed tasks
* Move completed tasks to the top of the list
* Change "Completed" button to "Remove"
* Delete completed tasks
* Dark/Light theme toggle 🌙☀️
* Dynamic theme icon switching

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## 📚 Concepts Practiced

* DOM Selection (`querySelector`)
* Event Listeners (`addEventListener`)
* Keyboard Events (`keydown`)
* Dynamic Element Creation (`createElement`)
* DOM Insertion (`append`, `prepend`)
* Class Manipulation (`classList.add`, `contains`, `toggle`)
* Conditional Logic (`if/else`)
* Dynamic UI Updates

## 🎯 What I Learned

While building this project, I gained hands-on experience with JavaScript DOM manipulation and learned how to:

* Create and insert elements dynamically
* Handle user interactions
* Build theme-switching functionality
* Update UI based on application state
* Implement task management features using JavaScript
